```bash
  yarn create vite
```

# Instruções para Baixar o projeto:

Para baixar as depêndencias do projeto:

Usando yarn:

```bash
  yarn
```

Usando npm:

```bash
  npm install
```

Para baixar a mesma versão do node desse projeto dê:

```bash
nvm use
```

Para rodar o projeto:

Usando yarn:

```bash
  yarn dev
```

Usando npm:

```bash
  npm run dev
```

# Observação: Para refazer o build do projeto, temos:

```bash
  npm run build

  firebase login

  firebase init

  firebase deploy

```
